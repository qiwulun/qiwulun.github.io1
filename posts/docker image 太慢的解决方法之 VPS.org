假设我们要下载的 image 是 phusion/baseimage:0.9.19

- 在远端 VPS 上
  - =docker pull phusion/baseimage:0.9.19=
  - =docker save phusion/baseimage /baseimage.tar=
- 在本地
  - =scp root@path_to_vps:/baseimage.tar ~/baseimage.tar=
  - =sudo docker load < ~/baseimage.tar=
