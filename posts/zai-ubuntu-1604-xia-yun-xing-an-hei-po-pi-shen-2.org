安装 wine

#+BEGIN_SRC bash
sudo apt-get install wine
#+END_SRC

切换到 Diablo II 的安装目录下
#+BEGIN_SRC bash
wine D2Loader.exe -w
#+END_SRC

如果要是想运行大箱子补丁

#+BEGIN_SRC bash
wine Plugy.exe -w
#+END_SRC

如果要是想运行大箱子 + hackmap 补丁

#+BEGIN_SRC bash
wine D2Loader.exe -w
#+END_SRC
